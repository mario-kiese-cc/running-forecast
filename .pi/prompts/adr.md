Create an Architecture Decision Record for: $@

1. Read `docs/decisions/001-adr-template.md` for the template format
2. Find the next available ADR number by listing `docs/decisions/`
3. Write the ADR with:
   - Clear context explaining the forces at play
   - At least 2 options considered with pros/cons
   - A decisive recommendation with rationale
   - Concrete consequences (what becomes easier AND harder)
4. Save to `docs/decisions/NNN-kebab-case-title.md`
